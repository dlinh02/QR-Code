# Plugin QR Code Payment Gateway For WooCommerce.

## Installation
1. Download the plugin.
2. Unzip the downloaded file and upload it to the /wp-content/plugins/ directory of WordPress.
3. Activate the plugin through the 'Plugins' menu in WordPress.

## Configuration
1. Visit https://my.vietqr.io
   - Create an account.
   - Go to User > Profile.
   - Copy the Client ID and API Key.
2. Go to WooCommerce > Settings > Payments.
3. Select QR Code and configure the necessary settings.
   - **Client ID**: Enter your Client ID.
   - **API Key**: Enter your API Key.
   - **Bank**: Select your bank.
   - **Account Number**: Enter your account number.
   - **Account Name**: This will be apparent after entering your correct account number.
